Search

Semantic and full-text search across the entire Hugging Face paper library. Find what you need without leaving the shell.

Read

Pull full paper content as clean Markdown. Perfect for reading in the terminal or piping into your favorite LLM for analysis.

Explore

Instantly discover linked models, datasets, and spaces associated with any paper. Bridge the gap between research and code.

Agent-ready

Native JSON output mode and a zero-dependency 7.8MB binary make it the perfect tool for AI agents and automation scripts.

Quick Install

go install github.com/zakelfassi/hfpaper@latest

Or download the pre-compiled binary for your architecture from GitHub Releases.
