What it does

Search. Read. Cite. Explore.

One binary wraps the entire Hugging Face Papers API. Semantic search, full paper markdown, BibTeX citations, linked models — all from the terminal.

Search

Semantic + full-text search across 500K+ papers. Natural language queries work.

Read

Full paper as clean markdown. Pipe to an LLM, save to disk, or read in the terminal.

Cite

Generate BibTeX, APA, or MLA citations from any paper ID. One command.

Explore

Find linked models, datasets, and Spaces for any paper. Bridge research to code.

For agents

MCP server included.

Run hfpaper mcp to start a Model Context Protocol server. Your agent can search, read, and cite papers natively. 7 tools, stdio transport.

"mcpServers": {
  "hfpaper": {
    "command": "npx",
    "args": ["-y", "hfpaper", "mcp"]
  }
}

Zero install. Paste this into Claude Desktop, Cursor, or Windsurf. That's it.
