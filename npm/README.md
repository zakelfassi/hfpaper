# hfpaper

AI research papers from your terminal.

Search, read, cite, and explore the full Hugging Face Papers ecosystem — with MCP server support for Claude, Cursor, and Codex.

## Install

```bash
npm install -g hfpaper
```

Or run without installing:

```bash
npx hfpaper search "vision language models"
```

## Usage

```bash
hfpaper search "multimodal reasoning" --limit 5
hfpaper read 2602.08025
hfpaper daily --trending
hfpaper cite 2602.08025 --format apa
hfpaper summary 2602.08025
hfpaper models 2602.08025
```

## MCP Server

```bash
hfpaper mcp
```

Works with Claude Desktop, Cursor, and any MCP client.
